1. 이해
- 땅(land)은 총 N행 4열로 이루어져 있고, 모든 칸에는 점수가 쓰여 있다.
- 1행부터 땅을 밟으며 한 행씩 내려올 때, 각 행의 4칸 중 한 칸만 밟으면서 내려와야 한다.
- 한 행씩 내려올 때, 같은 열을 연속해서 밟을 수 없다. 
- 땅은 2차원 배열로 주어진다. 
- 점수의 최대값을 구해야한다. 

2. 계획
