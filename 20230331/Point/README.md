문제(점 찍기)

정수 k와 원점과의 거리를 나타내는 정수 d가 주어졌을 때, 점이 총 몇 개 찍히는지 return 하는 solution 함수를 완성하세요.

---

1. 이해
- a, b는 제한이 없음
- 좌표(a*k, b*k)는 원점과의 거리가 d를 넘지 말아야함
- 거리 구하는 공식은 (a*k)^2 + (b*k)^2의 제곱근

2. 계획
- a 부터 숫자를 늘린다 (거리를 넘어서기 전까지)
- b를 늘린다 (거리를 넘어서기 전까지)
